Плагин для интеграции CMS Wordpress и CRM Мегаплана.

== Description ==

Плагин работает как дополнение к основному плагину Contact Form 7. Все формы, созданные через плагин, автоматически интегрируются с вашей CRM. Проверьте перед началом работы, что у вас установлен Contact Form 7, у вас есть аккаунт в CRM Мегаплан и вы зарегистрированы в <a href="http://mgtsk.ru" target="_blank">mgtsk.ru</a>
